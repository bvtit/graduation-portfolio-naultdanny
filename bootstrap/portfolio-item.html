Cisco I Final Project

Project Description

Design and implement a private networking solution for a multi-location small, stable commercial organization, which currently has 4 locations indicated in the diagram, all located in the United States.

Project Details

  • 55 Hosts in Tampa
  • 10 Hosts in Atlanta
  • 28 Hosts in Boston
  • Subnetting using VLSM and the network address 172.16.1.0/24

Given a diagram of a network topology designed for a company with multiple locations, I had to create a subnetting scheme for a network using VLSM. I then had to configure the routers and switches in the network to work correctly using the appropriate commands. I also had to include a purchase order of all the equipment which was used in this project, including routers, switches, and expansion cards, etc., and write a letter of intent to the company which the network was designed for.
